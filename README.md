## Setting up the app
```bash
1. Установить зависимости
$ npm install

2. Настроить порт и данные для базы данных (по видео)

3. Скопировать содержимое архива с фильмами 
по адресу *path_to_this_app*/src/movies

4. Подготовить базу данных (по видео)

5. Запустить приложение:
$ npm run start:dev

6. Отправить GET-запрос для наполнения БД (в консоли будет спамить фраза, 
1 фраза = 1 перенесённый фильм)
localhost:3000/load_info_into_database

7. После окончания процесса отправлять GET-запросы по эндпоинту ниже
```


## Running the app

```bash
# вам нужен этот режим
$ npm run start:dev
```

## Api

```bash
#GET-request
#Заполняет БД фильмами
http://localhost:3000/load_info_into_database
----------------------------------------------------


#GET-request
#Возвращает список фильмов с фильтрами по году
http://localhost:3000/movies

#Query:
year=2008
director=Джей Лоуи
- Можно комбинировать или вообще не использовать. 
- Во втором случае вернутся все фильмы, аккуратнее:)
----------------------------------------------------


#GET-request
#Возвращает фильм по его айди
http://localhost:3000/movies/21222

#Params:
id=21222
----------------------------------------------------


#GET-request
#Возвращает список всех жанров
http://localhost:3000/genres
----------------------------------------------------


#GET-request
#Возвращает персону по её айди
http://localhost:3000/person/1337

#Params:
id=1337
----------------------------------------------------
```
